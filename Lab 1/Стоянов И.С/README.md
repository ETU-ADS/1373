## Getting Started

Для запуска потребуется `python=3.10.2`

### Installing

1. Склонируйте репозиторий и запустите:

    ```bash
    python main.py
    ```

## Examples

```bash
>> python main.py
>> input: 3 + 4 * ( 2 - 1 )
>> output: 3 4 2 1 - * +
```

```bash
>> python main.py
>> input: 5 + 3 * 7
>> output: 5 3 7 * +
```

```bash
>> python main.py
>> input: 3 * ( sin ( 4 ) - 1 ) + 4 * ( cos ( 2 ) * 7 )
>> output: 3 4 sin 1 - * 4 2 cos 7 * * +
```
