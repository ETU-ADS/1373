## Формат MR для загрузки работ
1. Название ветки: <фамилия>_<lab1/lab2/lab3/course>
2. Содержание MR:  
- <выполняемая работа>/<Фамилия ИО>/
    - README.md - инструкция для запуска кода и дополнительная информация
    - Отчёт в формате .pdf
    - src/
        - Файлы с исходным кодом
